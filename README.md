# qt-research-terminal
A standalone quant analytics terminal built in a single HTML file. Implements a 3D implied volatility surface (Three.js + SVI/SABR), Black-Scholes-Merton Greeks engine (Delta, Gamma, Theta, Vega, Rho), cross-asset correlation matrix with Ledoit-Wolf shrinkage, and a Monte Carlo path simulator with Merton jump-diffusion. No backend. No framework.
